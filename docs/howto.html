    Main Program

    The installer puts a single icon on your desktop; click it to run the main program.

    The main program does two things:

    1. It provides some functions you can use right through the program (like rearranging buildings on your planets and building halls out of glyphs).
    2. It lets you set up preferences for how you want things done in your scheduled tasks, like autovoting and training spies.
    The programs that actually perform those scheduled tasks do get installed along with the main program, but they won't have icons on your desktop.

    Scheduled Programs

    These programs will be on your computer wherever you installed LacunaWax, which will probably be something like:
    C:\Program Files\TMT Tools\LacunaWaX\bin\
    (adjust as needed for your version of Windows).

    The scheduling programs available are:

    • Schedule_archmin.exe
      Handles both glyph pushing and automated searching of individual arch mins.
    • Schedule_autovote.exe
      Votes for SS propositions based on your settings
    • Schedule_lottery.exe
      Plays the lottery on your requested colonies
    • Schedule_train_spies.exe
      Trains spies in the skills you've chosen provided the spies are both available and currently sitting on a planet with the appropriate training building

    Setting up a run schedule

    If you already know how to do so, just use the Windows Task Scheduler or your favorite third-party scheduling program. But keep in mind that the scheduled programs will all open CMD windows when they're run, so be sure to set up your scheduled tasks to run hidden.

    If you're unfamiliar with setting up scheduled tasks, or your normal task scheduler does not provide the option to run scheduled tasks in a "hidden" mode, See the step-by-step guide on setting up scheduled tasks.
