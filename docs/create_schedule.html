    Set up Scheduled Tasks

    Remember that LacunaWaX itself just allows you to set up what you want to have happen when its scheduled tasks run - LacunaWaX itself does not schedule anything!

    So after running LacunaWaX and setting up your various preferences, you'll need to set up the scheduled tasks so that they run when you want them.

    This is a pain and has caused some confusion for several people.

    You can certainly use the built-in Windows Task Scheduler if you want to, but if you're not used to it, it can be a horror show to try to figure out.

    Instead, download and install System Scheduler Free from Splinterware. It's easier to use than the Windows scheduler and it will look the same across different versions of Windows, so you can just follow the instructions below step-by-step.

    Preferences

    After installing System Scheduler, you'll have a little alarm clock icon in your system tray (next to your clock). Right-click it and chose "Preferences", then chose the Startup/Logging tab.

    Make sure the "Do NOT start System Scheduler automatically on logon" checkbox is unchecked! After you've made sure of that, you can close the preferences window.

    Create new event

    Go back to the alarm clock icon in your systray again and just double-(left)-click it to bring up the actual scheduler.

    You can see I have my Arch Min and Spy Training already set up, but I don't have Autovote set yet. To set that up, click the indicated green circle icon to create a New Event.

    Name it and select the program to run

    Give your event a title, then click the yellow folder icon next to Application. Browse to your TMT Tools/LacunaWaX/bin/ folder and select "Schedule_autovote.exe".

    Make sure to run it Hidden

    Make sure to select the "Hidden" State. This will keep the black CMD window from popping up at you each time the scheduled event runs.

    Choose times for it to run

    I want my autovote to run every three hours. So I selected "Every Hour/Selected Minutes" in the dropdown box up top. Then, in the Hours box, I entered:

    00,03,06,09,12,15,18,21
    Also, I typed 30 in the Minutes box, so my task will run at 30 minutes past the hour, every three hours.

    If you wanted your task to run every two hours, it would be:

    00,02,04,06,08,10,12,14,16,18,20,22
    ...etc

    Save

    Don't forget to save your new scheduled task by clicking the floppy icon.

    Done!

    And that's it - you can see my new scheduled task has been set up, and it displays the time when it'll run next. You can now just close the System Scheduler window.
