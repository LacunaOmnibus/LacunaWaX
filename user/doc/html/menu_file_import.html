File... Import Preferences

Imports preferences and settings from an outside LacunaWaX database.

This allows you to pull in your previous settings if you've recently upgraded to a new version of LacunaWaX, or import from a backup in case your current database gets corrupted.

The main LacunaWaX database is a file called lacuna_app.sqlite, and it lives in the user/ directory (folder). So with an installer version of LacunaWaX on Windows, you'd be looking for something like C:\Program Files\TMT Tools\LacunaWaX\user\lacuna_app.sqlite.
