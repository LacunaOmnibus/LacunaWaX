Mission Editor Overview

  • Name - The name of your mission.
  • Max University Level - Players with a university level higher than this will not be able to access your mission.
  • Description - What the mission is all about; this is the story you're telling.
  • Network 19 Headline - What shows up in Network 19 when the mission appears.
  • Network 19 Completion - What shows up in Network 19 when the mission is completed.
    To include the name of the empire that completed the mission in Network 19, use "%s" as a placeholder, eg "SomeMission was completed by %s, who is very happy about the rewards."
