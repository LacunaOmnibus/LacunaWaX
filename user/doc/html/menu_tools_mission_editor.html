Tools... Mission Editor

Create your own missions and upload them for use in TLE.

As you work on a mission, you can Save it as often as you want, and then come back and work on it later.

When you're happy with your mission, Export it to a file, which you can then upload to the Missions forum.

Any mission can be Deleted at any time, but note that this will completely wipe it.
